# ESOUI.com Upload Information

This file contains information for uploading Beltalowda to ESOUI.com.

## Addon Information

- **Addon Name**: Beltalowda
- **Category**: PvP, Group Tools
- **Compatible Game Version**: Gold Road and later
- **API Version**: 101047, 101048

## Description for ESOUI.com

Beltalowda is a group PvP addon for Elder Scrolls Online designed to enhance group coordination and combat effectiveness in PvP scenarios.

### Features
- Group coordination tools for PvP
- Account-wide saved settings
- Compatible with latest ESO API versions
- Lightweight and performance-optimized

### Installation
The addon can be installed either manually or through the Minion app.

**Manual Installation:**
1. Download the latest version
2. Extract to your ESO AddOns folder
3. Enable in the ESO Add-Ons menu

**Minion App:**
1. Search for "Beltalowda" in Minion
2. Click Install

## Files Included

- Beltalowda.txt - Addon manifest
